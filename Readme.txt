WordPress Popup plugin helps you create custom popups with any content and style. Powerful Popup Builder with flexible settings and statistic.

== Description ==

Popup Generation - the simple and functional WordPress plugin for creating pop-ups. With the popup plugin, you can easily create modal windows, pop-ups and lightboxes for your own needs as you wish and then insert anywhere place on the site. 

Together with the form generation plugin, you can create a very strong tool for lead generation.

Together with the [Form Generation](https://wordpress.org/plugins/form-generation/) and [Button Generation](https://wordpress.org/plugins/button-generation/) plugins, you can create a very strong tool for [Lead Generation](https://wordpress.org/plugins/leadgeneration/).

= Can be used for: =
* Information modal windows;
* Facebook messenger;
* Popup with the map;
* Feedback form;
* Float block a with subscription;
* Info-Bar Top or Bottom;
* Interactive widgets;
* Unique user menus;
* Functional popup panels and much more.

= Features =
* Create any kind of modal windows with a powerful editor
* Style them how you want
* Place them anywhere
* Insert any content
* Analytics for each popup
* Tooltips to create popup

= General popup settings: =
* Choose the Triggers (Auto or Click)
* Animation Speed
* Set the cookie time

= Settings for closing popup =
* Click popup Overlay
* Press ESC

= Popup style settings =
* Choose popup color and background
* Location
* Width, Height and paddings
* Border settings
* Drop Shadow
* Content Font

= Close Button style settings =
* Choose type of the button: Icon or Text
* Location settings
* Border settings
* Color and background settings
* Font settings


== Installation ==

* Installation option 1: Find and install this plugin in the `Plugins` -> `Add new` section of your `wp-admin`
* Installation option 2: Download the zip file, then upload the plugin via the wp-admin in the `Plugins` -> `Add new` section. Or unzip the archive and upload the folder to the plugins directory `/wp-content/plugins/` via ftp
* Press `Activate` when you have installed the plugin via dashboard or press `Activate` in the in the `Plugins` list 
* Go to `Popup Generation` section that will appear in your main menu on the left
* Click `Add new` to create your first popup
* Setup your popup
* Click Save
